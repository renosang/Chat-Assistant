Hướng dẫn Lấy API Key Gemini

Để sử dụng tiện ích này, bạn cần cung cấp một API Key từ Google AI Studio. Vui lòng làm theo các bước sau:

  1. Truy cập Google AI Studio:
    Mở trình duyệt và truy cập: https://aistudio.google.com/app/apikey
  2. Đăng nhập:
    Đăng nhập bằng tài khoản Google (cùng tài khoản bạn dùng cho công việc nếu có).
  3. Tạo API Key mới:
    Trang web sẽ đưa bạn thẳng đến mục "API keys".
    Bấm vào nút + Tạo API key trong dự án mới (Create API key in new project).
  4. Sao chép (Copy) Key:
    Một popup sẽ hiện ra chứa API Key của bạn. Đây là một đoạn mã dài. Hãy bấm vào biểu tượng "Copy" (Sao chép) bên cạnh nó.
  5. Dán (Paste) Key vào Tiện ích:
    Quay lại trình duyệt, bấm vào biểu tượng của tiện ích (Extension) này trên thanh công cụ.
    Dán (Paste) đoạn mã bạn vừa copy vào ô "Nhập API Key" và bấm Lưu.
  6. Hoàn tất!
    Bạn đã sẵn sàng sử dụng.
⚠️ Bảo mật: API Key này giống như mật khẩu. Tuyệt đối không chia sẻ nó cho người khác hoặc đăng lên mạng.
